# Smart City Air Quality Monitor (MQTT)

Projeto acadêmico utilizando ESP32 e MQTT para monitoramento da qualidade do ar.

## Objetivo

Monitorar níveis de poluição em ambientes urbanos.

## Componentes

- ESP32
- MQ-135
- LED
- Resistor

## Funcionamento

O sensor MQ-135 lê a qualidade do ar.

O ESP32 envia os dados via MQTT.

O LED indica quando a qualidade do ar está ruim.

## MQTT

Broker: broker.hivemq.com  
Tópico: smartcity/airquality
